# projects/

What's actively being built. One subfolder per project.

Each project folder accumulates:

- `vision.md` — the original plain-English vision(s), dated.
- `spec-vN.md` — generated specs, versioned, each with its Evidence Standard
  tags and risk register intact.
- `decisions.md` — the user's approvals/rejections at each gate, so the
  history of "why" survives.
