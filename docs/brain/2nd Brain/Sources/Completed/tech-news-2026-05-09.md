# AI + Tech News Intelligence Brief — 2026-05-09

## Anthropic / Claude Updates (Week of May 5–9, 2026)

### Code with Claude 2026 Conference — May 6, 2026 (San Francisco)

Source: Simon Willison live blog; 9to5Mac; MacObserver; SiliconANGLE.

Anthropic held its second annual Code with Claude developer conference on May 6, 2026. Three major features were announced for Claude Managed Agents, all entering beta/preview this week:

**1. Dreaming (Research Preview — launched May 7, 2026)**
- Claude agents can now review their own past sessions to identify patterns, extract insights, and self-improve between tasks
- Two modes: automatic memory updates, or manual human review before implementation
- Architecture: Scheduled process runs between active sessions; reads memory stores; curates and prunes memories over time
- Described as "self-improving agents" when combined with the existing Memory capability
- Netflix cited as an early deployer — using multi-agent orchestration for its platform team

**2. Outcomes (Public Beta — launched May 7, 2026)**
- Users define a success rubric describing what "done" looks like
- A separate grader evaluates output against the rubric in its own context window (isolated from the agent's reasoning chain — no contamination)
- When the grader marks output as insufficient, it pinpoints the gap and the agent iterates
- Webhook notifications fire on completion — enabling async pipeline integration

**3. Multi-Agent Orchestration (Public Beta — launched May 7, 2026)**
- A lead (orchestrator) agent breaks large jobs into discrete subtasks
- Specialist sub-agents run in parallel on a shared filesystem
- Enables fleet-scale task execution; replaces sequential chaining with true parallelism

### Claude Opus 4.7 — General Availability

- Opus 4.7 is now generally available (previously in limited beta)
- Improvements: Software engineering benchmarks, complex long-running coding tasks, higher-resolution vision
- API rate limits doubled across Pro, Max, Team, and Enterprise tiers
- Peak-hour rate reductions removed for Pro and Max plans
- Opus API limits raised on new SpaceX compute capacity

### Claude Code — Latest Release (May 2026)

Source: Releasebot; Claude Code GitHub releases; code.claude.com/docs.

Key updates shipped this week:
- `--plugin-url` flag: Load a plugin .zip archive from a URL for the current session (no local install required)
- `CLAUDE_CODE_PACKAGE_MANAGER_AUTO_UPDATE` env variable: Enables automatic background updates via Homebrew or WinGet
- PostToolUse output replacement functionality (new hook capability)
- Fullscreen scroll fixes; scrollable dialogs; wrapped URL clicking
- SDK fork subagent support in non-interactive sessions
- Fixes: Sessions, editors, cache handling, OAuth, MCP, VS Code integration
- Plugin URL loading + smarter package-manager auto-updates + new controls for sync output and gateway model discovery
