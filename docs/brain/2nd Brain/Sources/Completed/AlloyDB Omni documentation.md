AlloyDB Omni is a downloadable database software package that lets you deploy a streamlined edition of AlloyDB for PostgreSQL in your own computing environment. AlloyDB Omni provides enterprise-grade performance and availability while maintaining 100% compatibility with open-source PostgreSQL.

To deploy AlloyDB Omni in your computing environment, see [Choose your deployment environment](https://docs.cloud.google.com/alloydb/omni/docs/choose-deployment) or select one of the following:

- [Install AlloyDB Omni using containers](https://docs.cloud.google.com/alloydb/omni/containers/current/docs/deploy-vm)
- [Install AlloyDB Omni using the container orchestrator](https://docs.cloud.google.com/alloydb/omni/kubernetes/current/docs/deploy-kubernetes)
- [Install AlloyDB Omni using RPM](https://docs.cloud.google.com/alloydb/omni/linux/current/docs/deploy-vm)
- [Install AlloyDB Omni using the RPM orchestrator](https://docs.cloud.google.com/alloydb/omni/redhat-orchestrator/current/docs/install-orchestrator)

Access AlloyDB Omni on [Docker Hub](https://hub.docker.com/r/google/alloydbomni/tags), [Google Artifact Registry](https://gcr.io/alloydb-omni/alloydbomni), [Google Cloud Marketplace](https://docs.cloud.google.com/marketplace/docs), or [AWS Marketplace](https://aws.amazon.com/marketplace/pp/prodview-sne7ttiby7nxi).

AlloyDB Omni follows PostgreSQL versioning. See [Database version policies](https://docs.cloud.google.com/alloydb/omni/docs/db-version-policies) for more information.
