# Marketing Brain — SEO Operating System

Karpathy-style Hot / Index / Wiki pattern for SEO operations. An Obsidian vault + Claude Code (or Codex) skill that converts DataForSEO research into a 30/60/90-day execution plan and compounds with every run. Direct analogue to [[2nd Brain build|this very wiki]] but pointed at organic search rather than the empire.

## Source

`Sources/Marketing Brain My AI SEO System Walkthrough (Claude Code + Obsidian).md` — Agrici Daniel, AI Marketing Hub Pro, 2026-05-08.

Companion repos: `claude-seo`, `claude-blog`, `claude-ads`, `flow` framework. Engine: `rankenstein.pro`.

## What it produces (the ULTIMATE BEAST plan)

Run-once-per-client output:

1. **Scaffolded Obsidian vault** — overview, client metadata, business-type template (affiliate / e-commerce / B2B / local SEO / services / publisher / SaaS).
2. **18,000+ unique keyword workbook** (typical — pulled from top 10 competitors' ranking sets, deduped). Columns: keyword, topic cluster, search volume, CPC, competition, KD, intent, SERP features, best competitor + URL.
3. **Top 100 highest-volume keywords mined for People-Also-Ask** → niche question map.
4. **Cannibalisation ledger** — keyword → URL map prevents two pages competing.
5. **Decisions log** — affiliate-link hygiene, booking attribution, citation strategy, consolidated guide vs. cluster, etc.
6. **30 / 60 / 90-day action plan** — day 0 setup (GSC, RPM source, affiliate baseline) → day 1–5 → 6–12 → 13–18 → month plan.

## Six-step pipeline

1. Find top 10 competitors via DataForSEO SERP.
2. Pull every keyword each competitor ranks for.
3. Build XLSX workbook; mine SERP for top 100 by volume.
4. Pull PAA + niche question expansion.
5. Scaffold Obsidian vault from template; populate client metadata + business type overlay.
6. Synthesise the BEAST plan against the FLOW framework (Find → Leverage → Optimise → Win — the 2025/2026 successor to the 2022/2023 ski-slope hub-pillar-cluster strategy).

## Why it compounds

Every run adds keywords, PAA entries, decisions, and audit findings to the vault. Vault grows with the project. Token-efficient because the template + SOP do the heavy reasoning — Claude only fills gaps. Typical full run: ~30 min – 1 h; $0.50–$5 DataForSEO API cost (default cap $5).

## Requirements

- Obsidian (free)
- Claude Code or Codex CLI subscription (Codex and Claude rated strongest; Gemini works but lags)
- DataForSEO API key

## Validated by Unichannel SEO ($325M exit)

Source: `Sources/Unichannel SEO with David Quaid…` (Quaid, head of SEO at Rapid7-acquired insights). Reinforces the thesis behind running Marketing Brain across the portfolio:

- For B2B / SaaS / tech, SEO routinely accounts for 75–90%+ of leads (LinkedIn 0.1% in one client G4 sample).
- Insight digital: SEO 6.9% of closed-won (April 2020) → 70% (Sept 2021) → acquired by Rapid7 for $325M (Nov 2021). Bid budget was $3k/month paid.
- Playbook: cornerstone pages targeting paid-search high-CPC terms (>$70 bids), reverse-engineering each into a glossary → what-is → how-to ladder. Plus paid-traffic remarketing that fills missing touch-points (15% conversion uplift on $10–$100k deals).
- Start link-building day 1 (holding page). "Wait and see" is the most common mistake — time alone earns nothing; activity does. DA is the wrong metric — prefer pages-with-traffic and referral traffic.
- Local Meetup-group ownership is a passive long-term backlink generator. Phill's parallel: ANZ industry-association events under [[industry-association-vision-2026]] should publish do-follow community pages.
