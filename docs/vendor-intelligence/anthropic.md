### Compliance surface distinction

Access Transparency (`anthropic.platform.access-transparency`) is a human-access
audit facility covering specific Anthropic **API** customer-data surfaces only.
It is **not** Claude Max usage telemetry and does not cover consumer or
application surfaces (Claude.ai, Claude Code local sessions, mobile/desktop
apps). Changes classify strictly under the API-compliance surface.
