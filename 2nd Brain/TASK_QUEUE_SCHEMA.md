## Status transitions

`queued -> claimed -> running -> artifact_ready -> approval_pending -> approved/rejected -> done`

Failure transition:

`running -> failed -> platform_improvement_task -> blocked/retry`

## Required fields by task type

| Type | Required evidence | Typical worker |
|---|---|---|
| research | citations, confidence, missing evidence | research-bi-worker |
| build | tests/build logs, patch artifact | build-worker |
| qa | validation commands and results | build-worker |
| seo | URLs/queries/schema/content evidence | research-bi-worker |
| growth | business evidence and expected impact | research-bi-worker |
| dashboard | status source and actionability | command-node |
| shipit | readiness checklist and blockers | build-worker |
